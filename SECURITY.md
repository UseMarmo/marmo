# Security Policy

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Report vulnerabilities by email to **security@usemarmo.xyz**. Include as much detail as possible: the component affected, steps to reproduce, and the potential impact.

We will acknowledge your report within 48 hours and aim to release a fix within 14 days for critical issues.

## Scope

- Co-signer server (`src/`)
- Smart contracts (`contract/`)
- Desktop app (`desktop/`)
- SDKs (`sui-sdk/`, `base-sdk/`)

## Supported Versions

Only the latest release on `main` receives security patches.
